Minimal RAG From Scratch

A Retrieval-Augmented Generation (RAG) pipeline built from scratch in Python without frameworks.

Features
- Load PDF documents
- Extract text page-wise
- Paragraph-aware chunking
- Generate embeddings using Sentence Transformers
- Store vectors in FAISS
- Retrieve the most relevant chunks for a user query
- Display retrieved context with similarity scores

Current Version
This project currently demonstrates the retrieval pipeline. The LLM generation step has been disabled because the focus is on understanding the retrieval process from first principles.
